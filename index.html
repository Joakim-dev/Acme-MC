This Project

Project Goal

The goal of this project was to make it rather casual but still nice. I wanted to implement some different styles, like the diagonal page, the gallery, the news and so on.Since my last project was build desktop first, I decided to do this one mobile-first.

Another very important goal was to improve the way the code was written, compared to the last project.


Workflow

I actually made the design for desktop first and from that, I developed the mobile design.

This led to some differences in the order of appearance between for example images and text.


Outcome

When testing the page runs as expected and the components are in place. More features could obviously be added, but the goal was to keep it fairly simple. I would say that the outcome of this project meets the goals well,

Audits: 96, 100, 100, 100

The 96 is because of large images, but I don't want to compress them further because of the loss of quality this would bring.
